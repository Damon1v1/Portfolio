# Portfolio
## Mobile Responsive Portfolio

## Features

* Consistent navbar header on each page
* Responsive layout
* Responsive images
* Working links
* Home page, portfolio page, and contact page
* Utilizes bootstrap components and grid system
* Sticky footer
* Functioning contact form
